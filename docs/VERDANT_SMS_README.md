## ğŸš€ Installation Guide

### Prerequisites

- **PHP**: 8.0 or higher
- **MySQL**: 8.0 or higher
- **Web Server**: Apache (XAMPP/LAMPP) or Nginx
- **Browser**: Modern browser with PWA support (Chrome, Edge, Firefox)

### Step-by-Step Setup

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Chrinux-AI/School_Management_System.git
   cd School_Management_System
   ```

2. **Import Database**

   ```bash
   mysql -u root -p < database/verdant-sms-schema.sql
   ```

3. **Configure Database**
   Edit `/includes/config.php`:

   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'school_management');
   define('DB_USER', 'root');
   define('DB_PASS', '');
   ```

4. **Start Web Server**

   - **XAMPP/LAMPP**: Place folder in `htdocs/`
   - Access: `http://localhost/attendance/`

5. **Create Default Admin**
   Run setup script:

   ```bash
   php create-default-admin.php
   ```

6. **Login**
   - **Super Admin**: `superadmin@verdant.edu` / Password: (set during setup)
   - **Demo Accounts**: See "Demo Login Credentials" section below
